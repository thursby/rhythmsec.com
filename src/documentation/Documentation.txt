Equalizer - responsive magazine template

Equalizer is a perfect template for music, events, culture, cinematic and games websites.

## HTML

Each template page validates correctly in the W3C validator and is fully compatible with HTML5.

## CSS

1. All CSS styles are included in css/styles.min.css.
2. Font Awesome is defined in css/font-awesome.min.css.
3. All CSS files are available in both versions - uncompressed and minified.

## Editing template

If you want to add/edit/remove some HTML elements you can do it quickly in the standard way in accordance with the philosophy of Bootstrap 3.

If you want to edit CSS styles, you can create additional CSS file with your own CSS code and include this file in page that needs it.

## JavaScript

All the custom JavaScript code is included in js/custom.js (js/custom.min.js) file. You can add your own code there.
